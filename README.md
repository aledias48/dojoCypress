# dojoCypress

# Automação do Site Centauro em Cypress

- Criar o projeto:
npm init -y

- Baixar o cypress:
npm install cypress

- Rodar o cypress:
npx cypress open

- Para reconhecer os métodos do cypress, inserir na primeira linha do codigo:
/// <reference types="cypress" />
